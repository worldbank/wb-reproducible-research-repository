A high-quality `README` file is the most important part of a reproducibility package. It is the entry point for any user and must clearly explain the project's purpose, contents, and instructions for reproducing the findings.

This page provides the necessary resources to create a comprehensive and clear `README` file for your research project.

---

### **Key Resources**

| Resource | Description |
| :--- | :--- |
| [**▶️ README Automation Tool**](https://dime-worldbank.github.io/wb-reproducible-research-repository-automation/) | An interactive tool that guides you through creating a complete `README` file by asking a series of questions about your project. |
| [**📄 README Template (.docx)**](https://github.com/worldbank/wb-reproducible-research-repository/raw/refs/heads/main/resources/README_Template.docx) | A Microsoft Word document template that you can download and fill in with your project's details. |
| [**📝 README Template (.md)**](https://github.com/worldbank/wb-reproducible-research-repository/blob/main/resources/README_Template.md) | A Markdown template, ideal for use in GitHub repositories and other plain-text environments. |
